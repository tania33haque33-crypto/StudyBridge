# StudyBridge - Study Abroad Platform

A comprehensive MERN stack platform for international students seeking education opportunities worldwide.

## 🚀 Features

### Authentication & Security
- ✅ JWT-based authentication
- ✅ OAuth integration (Google, Facebook, LinkedIn)
- ✅ Two-factor authentication (2FA)
- ✅ Email verification
- ✅ Password reset
- ✅ Role-based access control (Student, Admin, Moderator)

### University Management
- ✅ Advanced search with 15+ filters
- ✅ Detailed university profiles
- ✅ CSV import with image URL handling
- ✅ University comparison tool
- ✅ Save favorite universities
- ✅ AI-powered recommendations

### Scholarship System
- ✅ Comprehensive scholarship database
- ✅ Advanced filtering
- ✅ Deadline alerts
- ✅ Personalized recommendations
- ✅ Save scholarships

### Application Tracking
- ✅ Multi-status tracking
- ✅ Document upload
- ✅ Deadline reminders
- ✅ Timeline visualization
- ✅ Eligibility checker
- ✅ Document checklist generator

### Review System
- ✅ Multi-category ratings
- ✅ Photo uploads
- ✅ Admin moderation
- ✅ Verified reviews
- ✅ Helpful marking
- ✅ Report functionality

### Visa Guides
- ✅ Country-specific guides
- ✅ Financial calculators
- ✅ Document checklists
- ✅ Post-study work visa info

### Admin Panel
- ✅ User management
- ✅ University management
- ✅ Review moderation
- ✅ Analytics dashboard
- ✅ CSV import/export
- ✅ System settings

### Additional Features
- ✅ Real-time notifications (Socket.io)
- ✅ Email service
- ✅ File uploads
- ✅ Responsive design
- ✅ SEO optimized
- ✅ Automated cron jobs

## 🛠️ Tech Stack

### Backend
- Node.js & Express.js
- MongoDB Atlas
- JWT Authentication
- Passport.js (OAuth)
- Socket.io (Real-time)
- Nodemailer (Email)
- Multer (File Upload)
- CSV Parser
- Sharp (Image Processing)
- Cron Jobs

### Frontend
- React.js (Vite)
- Material-UI
- React Router
- React Query
- Zustand (State Management)
- Socket.io Client
- Axios
- React Hook Form
- Recharts (Analytics)
- Framer Motion

## 📋 Prerequisites

- Node.js (v16 or higher)
- MongoDB Atlas account
- Gmail account (for email service)
- OAuth credentials (Google, Facebook, LinkedIn)
